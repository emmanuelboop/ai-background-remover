# AI Background Remover

An AI-powered background remover built with React, FastAPI, and rembg.

## Live Demo

Frontend:
https://ai-background-remover-black.vercel.app

Backend:
https://ai-background-remover-production-6371.up.railway.app

---

## Features

- Upload images
- AI background removal
- FastAPI backend
- React frontend
- Railway deployment
- Vercel deployment

---

## Tech Stack

Frontend:
- React
- TypeScript
- Vite

Backend:
- FastAPI
- Python
- rembg
- Uvicorn

Deployment:
- Vercel
- Railway

---

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
